### Prerequisites
Below noted things you need to install to run this project in your system

- Node.js
- NPM
- MongoDB

### To Setup
Clone or download this repository

1. `cd react-app/server`
2. `npm install`
3. `cd react-app/client`
4. `npm install`

### To Run
To run node server
1. `cd react-app/server`
2. `node server.js`

To run react client
1. `cd react-app/client`
2. `npm start`
